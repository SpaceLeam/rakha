# CV_REIKHAL

Website Curriculum Vitae (CV) **Reikhal Akbar Damanik** — dibuat dengan HTML & CSS.

🌐 **Live:** https://spaceleam.github.io/CV_REIKHAL/

## Halaman
- **Beranda** (`index.html` / `cv_rad.html`) — profil & kegiatan sebagai Engineer Telekomunikasi
- **About Me** (`About Me.html`) — perkenalan singkat
- **Education** (`Education.html`) — riwayat pendidikan
- **Experience** (`Experience.html`) — pengalaman kerja
- **Contacs** (`Contacs.html`) — kontak (email, telepon, LinkedIn, Instagram)

## Teknologi
- HTML5
- CSS3
- Hosting: GitHub Pages
